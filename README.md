# Self-Driving-Car

Bu proje, JavaScript ile oluşturulmuş temel bir otonom araç simülasyonudur. Araç, sensör verilerini kullanarak yol üzerinde kendi kendine hareket edebilecek şekilde tasarlanmıştır.


## Özellikler
- Sensör (ray casting) ile çevre algılama  
- Basit bir yapay sinir ağı ile karar verme  
- Sinir ağı görselleştirme
